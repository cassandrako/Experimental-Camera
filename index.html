The Limitations of AI

In this project, I am using the Google Photo Database API and the Dalle API to create a cross-study on the limitations of AI art.

How does it work?

In Visual Studio Code, I created a new project containing several .json files for my APIs, two Javascript files, an HTML file and a CSS file. I also added an image folder to store all of the photos created through the Dalle API. Below, you will see the comparative study on how AI is able to create artwork to an extent. I create 2 columns in hopes to show a direct visualization of the images seen on Google, vs the images created from Dalle's art generation.
